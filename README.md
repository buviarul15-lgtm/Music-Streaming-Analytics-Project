# 🎵 Music Performance & Streaming Analytics Dashboard

## 📌 Project Overview
The **Music Performance & Streaming Analytics** project focuses on analyzing large-scale music streaming data across multiple digital platforms. Using **Microsoft Excel** and **Power BI**, raw data is transformed into meaningful insights through data cleaning, exploratory analysis, and interactive dashboards.

This project helps understand artist performance, song popularity, platform engagement, and streaming trends to support data-driven decision-making.

---

## 🎯 Objectives
- Clean and standardize raw music data
- Handle missing values and remove duplicates
- Validate metadata using ISRC where applicable
- Analyze:
  - Artist performance
  - Song performance
  - Album trends
  - Platform engagement
- Build an interactive Power BI dashboard
- Generate actionable business insights

---
